# OpenMesa

## 📌 Descrição do Projeto
O OpenMesa é um sistema desenvolvido com o objetivo de facilitar o controle de fichas de refeições em um restaurante popular.

O sistema substitui o processo manual de contagem, permitindo o registro digital em tempo real, reduzindo erros e tempo de fechamento.

---

## 🎯 Objetivo
Desenvolver um sistema simples, rápido e acessível para registrar a saída de refeições, melhorando a eficiência operacional da instituição.
